# My Blog Project

* SpringBoot를 통해 CRUD 게시판을 만드는 토이 프로젝트입니다. (Ongoing)

<br>

## 프로젝트 개발 환경 :: Back-End

* Java 8
* Gradle 4
* Spring Boot 2.3.2.RELEASE
* Spring Data JPA
* WebTestClient
* Docker
* MariaDB

<br>

## 프로젝트 개발 환경 :: Front-End

* Thymeleaf
* Bootstrap
